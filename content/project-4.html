Heat Index

This project explores a century of global temperature anomalies using robust spatial mapping to uncover the rapidly increasing thresholds of planetary warming.

Technical Documentation

Data Sources

  • NOAA Global Surface Temperature Anomalies (1880-2024).
  • NASA Earth Observatory Dataset parsing.

Methodology

The dataset was meticulously cleansed utilizing a robust Python backend pipeline comprising of Pandas and NumPy for dropping standard deviations below the critical noise floor threshold bounds.

Tableau Operations

  • Calculated fields using dual-axis overlay maps.
  • Dynamic filtering for time-series continuous playback.
  • Color palettes rigorously constrained to the Viridis heatmap gradient matrix for maximal visual accessibility.
